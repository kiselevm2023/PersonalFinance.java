# Приложение для бухгалтерского учета .

## Описание

Приложение для бухгалтерского учета написано на языке  Java. Приложение должно работать с двумя видами файлов-отчётов — месячными и годовым.  Для реализации графического интерфейса использовал библиотеку SWING. 
Данные хранил в файлах, загрузка идет с использованием библиотеки jfree. Приложение многоязычное.

Java, Java Swing , библиотека jfree .
